This is the Adafruit GPS library - the ultimate GPS library for the ultimate GPS module!

Tested and works great with the Adafruit Ultimate GPS module using MTK33x9 chipset ---—> http://www.adafruit.com/products/746 Pick one up today at the Adafruit electronics shop and help support open source hardware & software! -ada

◆ PMTK_SET_NMEA_UPDATE_100_MILLIHERTZ

#define PMTK_SET_NMEA_UPDATE_100_MILLIHERTZ   "$PMTK220,10000*2F"

Once every 10 seconds, 100 millihertz.

Different commands to set the update rate from once a second (1 Hz) to 10 times a second (10Hz) Note that these only control the rate at which the position is echoed, to actually speed up the position fix you must also send one of the position fix rate commands below too.
